# Homebrew Tap

This is a Homebrew tap for applications by David Medvedev.

## Installation

```bash
brew tap dodabuilt/tap
brew install --cask animated-wallpaper
```

## Available Casks

| Cask | Description |
|------|-------------|
| `animated-wallpaper` | Transform your macOS desktop with animated video and GIF wallpapers |

## Updating

```bash
brew update
brew upgrade --cask animated-wallpaper
```

## Uninstalling

```bash
brew uninstall --cask animated-wallpaper
```
